# 2025 Machine Learning (COSE362) Term Project: Binary Classification using Bank Marketing data

고려대학교 보건과학대학 바이오의공학부 2021250031 정예준

## Project Description
This project focuses on the **Bank Marketing** dataset, which contains data from direct marketing campaigns (phone calls) of a Portuguese banking institution.

The objective is to develop a **classification model** to predict whether a customer will subscribe to a term deposit.

## Dataset Summary
| Item | Description |
|------|-------------|
| Dataset Characteristics | Multivariate |
| Subject Area | Business |
| Associated Task | Classification |
| Feature Type | Categorical, Integer |
| Number of Instances | 45,211 |
| Number of Features | 16 |

## Target
- **Target Variable:** `y`
- **Prediction Goal:** Determine whether a client will subscribe to a term deposit

## Objective
The purpose of this project is to analyze customer and campaign data and build a predictive model for term deposit subscription.
